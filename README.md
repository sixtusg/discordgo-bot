# sixtus-discord-bot
Creating a Discord bot using Discord's API and the DiscordGo library for general use in guilds.

## Usage
Run `./main -t [TOKEN_GOES_HERE] -p [PREFIX_GOES HERE]`
(Note that this is for usage in packages.)
